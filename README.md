# 🐝 꿀템(Ggultem) Frontend

React 기반 중고마켓 및 커뮤니티 웹 서비스의 프론트엔드입니다.  
백엔드(Spring Boot)와 REST API 통신을 통해 데이터를 처리하고 화면에 출력합니다.

---

📅 개발 기간: 2026.03 ~ 2026.04 (팀 프로젝트)

---

## 🛠 기술 스택
- React, JavaScript
- Axios, Redux Toolkit
- HTML, CSS

---

## 🚀 주요 기능
- 게시글 목록 및 상세 화면 UI 구현
- 댓글 및 대댓글(계층형) UI 구현
- 검색 기능 UI 구현
- 관리자 페이지 UI 구현

---

## 🔗 백엔드 연동
- REST API를 통해 게시글, 댓글, 검색 데이터 처리
- Axios를 활용한 비동기 통신 구현

---

## 📁 프로젝트 구조

- Pages: 화면 단위 컴포넌트
- Components: 재사용 UI 컴포넌트
- API: 서버 통신 로직 관리
